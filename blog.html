What is event delegation in the context of the DOM, and why is it useful?

Event delegation in JavaScript is a pattern that efficiently handles events. Events can be added to a parent element instead of adding to every single element. It is useful because the event can be listened to on multiple elements by using just one vent handler. It also uses less memory and gives better performance.

How do you manipulate an element's attributes and styles using the DOM?

Answer:
Attributes manipulation is used to directly alter the characteristics of HTML elements.
Styles manipulation changes the visual representation of elements either by adjusting individual properties or applying CSS classes.
